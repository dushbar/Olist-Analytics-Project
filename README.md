# Olist E-Commerce Analytics Project

## Project Objective
Analyze the Olist Brazilian e-commerce marketplace dataset to understand sales performance, delivery reliability, customer satisfaction, customer retention, and seller risk.

## Tools Used
- Python
- pandas
- SQL Server
- Power BI

## Dataset
Olist Brazilian E-Commerce Public Dataset.

## Workflow
1. Cleaned and transformed raw CSV files using Python.
2. Loaded cleaned data into SQL Server staging tables.
3. Created SQL analytical views for reporting.
4. Built Power BI dashboard pages for executive, delivery, review, retention, and seller analysis.

## Key Analyses
- Sales and revenue trends
- Category performance
- Delivery delay analysis
- Review score impact
- Customer cohort retention
- Seller risk segmentation

## Key Findings
- Late delivery is associated with lower review scores and higher low-review rates.
- Some states show both high late-delivery rates and low review scores.
- Repeat customer behavior is weak, with approximately 3% of customers placing more than one delivered order.
- Some high-revenue sellers also show weak review or delivery performance.

## Dashboard Pages
1. Executive Overview
2. Delivery Performance
3. Review Impact Analysis
4. Cohort Retention
5. Seller Performance & Marketplace Risk

## Business Value
This project demonstrates an end-to-end analytics workflow: data cleaning, relational modeling, SQL analysis, dashboarding, and business insight generation.
